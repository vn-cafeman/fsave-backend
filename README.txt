📦 HƯỚNG DẪN TRIỂN KHAI FSAVE CLONE (Node.js + yt-dlp)

1️⃣ Cài đặt Node.js (nếu chưa có):
→ https://nodejs.org

2️⃣ Cài đặt yt-dlp:
- Trên Linux/macOS: sudo pip install -U yt-dlp
- Trên Windows: tải yt-dlp.exe từ https://github.com/yt-dlp/yt-dlp/releases

3️⃣ Giải nén thư mục này, mở terminal và chạy:
   npm install

4️⃣ Khởi chạy ứng dụng:
   node server.js

5️⃣ Mở trình duyệt:
   http://localhost:3000

✅ Dán link video TikTok, Facebook, YouTube, Instagram... và tải về dễ dàng.

📌 Lưu ý: Một số nền tảng có thể chặn IP, nên dùng VPN nếu cần.
